Guia de lançamento

Colocar o Maxled CRM no ar

Dois cadastros gratuitos, sem cartão de crédito. Depois disso, toda vez que eu enviar código novo, o site atualiza sozinho — você só acessa o link.

1

Criar o banco de dados (Neon)

Onde os dados do CRM (clientes, negócios, etc.) ficam guardados.

  1. Acesse neon.tech e clique em Sign up → Continue with GitHub.
  2. Crie um projeto novo — qualquer nome, ex: maxled-crm.
  3. Na tela do projeto, copie a Connection string (começa com postgresql://).
  4. Cole essa string aqui na nossa conversa — eu configuro o banco de dados remotamente para você (crio as tabelas e o usuário mediador inicial).
Essa string tem uma senha dentro — é normal, ela é só para o banco de dados e ninguém além de nós dois vê essa conversa.
2

Publicar o site (Vercel)

Onde o CRM em si roda.

  1. Acesse vercel.com → Continue with GitHub.
  2. Add New → Project → selecione o repositório Maxled.
  3. Antes de clicar em Deploy, abra Environment Variables e adicione as três abaixo (copie e cole os valores):
DATABASE_URL a connection string do Neon (passo 1)
AUTH_SECRET OMc+c0eMVnM0ubqS14alp35NzBaCeFJLuIdLAtSQVlI=
NEXTAUTH_URL deixe em branco por enquanto
  1. Clique em Deploy e aguarde (leva 1–2 minutos).
3

Último ajuste

Depois que o deploy terminar, a Vercel te dá um endereço (ex: maxled-crm.vercel.app).

  1. Copie esse endereço.
  2. Cole ele aqui na nossa conversa também — eu te aviso exatamente onde colar de volta como NEXTAUTH_URL (com https:// na frente) e você clica em Redeploy uma última vez.
A partir daqui, é só isso — nunca mais precisa mexer em nada de hospedagem.
Dúvidas em qualquer passo — cole o print ou a mensagem de erro na conversa.
